# Sistema de Casa de Empeño

Aplicación full-stack para la gestión de una **casa de empeño**, que permite administrar clientes, empeños, pagos y control de inventario.

## Tecnologías utilizadas

### Backend

* Java
* Spring Boot
* API REST
* Base de datos relacional

### Frontend

* Flutter
* Dart

## Estructura del proyecto

```
sistema-empeno-app
│
├── Backend
│   └── API desarrollada con Spring Boot
│
└── frontend
    └── Aplicación móvil desarrollada con Flutter
```

## Funcionalidades principales

* Registro de clientes
* Registro de empeños
* Control de artículos empeñados
* Gestión de pagos
* Consulta de historial

## Instalación

### Backend

1. Clonar el repositorio
2. Abrir el proyecto en IntelliJ IDEA
3. Ejecutar la aplicación Spring Boot

### Frontend

1. Abrir el proyecto en Visual Studio Code
2. Ejecutar:

```
flutter pub get
flutter run
```
